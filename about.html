Our Mission

What is Speakly?

Speakly is a web-based English learning platform designed to help students improve their speaking skills through conversations, vocabulary practice, grammar lessons, and quizzes.

Built as a final project for the English Conversation 2 course, Speakly aims to make English learning more interactive, accessible, and enjoyable for Indonesian students.

The platform focuses on real-world communication scenarios — from campus life to job interviews — so students can practice English they'll actually use.

🎯

Project Objectives

To create an interactive, modern English learning website that helps students practice conversation, vocabulary, grammar, and self-assessment through quizzes in a single platform.

🛠️

Technology Used

Built entirely with HTML5, CSS3, and Vanilla JavaScript — no frameworks, no backend, no database. A pure frontend project demonstrating the power of modern web standards.

📚

Course Context

This platform was developed as the final project for English Conversation 2 at the Teknologi Rekayasa Perangkat Lunak study program, demonstrating both language and technical skills.
